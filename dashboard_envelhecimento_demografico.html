METODOLOGIA
Idade apurada na referência de cada snapshot. Séries anuais usam a última referência de cada ano. O corte de 55 anos é proxy de proximidade de aposentadoria; a projeção formal de desligamentos está no painel de BI da SGP.
PROTEÇÃO DE DADOS · LGPD
Este painel exibe exclusivamente estatísticas agregadas. Nenhum dado identificado ou identificável é publicado: células com menos de 5 registros são suprimidas ou agregadas (k-anonimato, k=5); raça/cor é divulgada apenas em grupos agregados; dados de deficiência, saúde e identidade de gênero não são exportados para esta página. A base de origem não é versionada nem publicada.
